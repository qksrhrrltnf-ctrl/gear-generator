# 기어 설계 도구

평기어 쌍과 랙 피니언을 설계하고, 화면에서 맞물림을 확인한 뒤 SVG/DXF/STL 파일로 내보내는 교육용 React 웹앱입니다.

## 실행

```bash
npm install
npm run dev
```

## 검증

```bash
npm run lint
npm test
npm run build
```

## 주요 기능

- 평기어 쌍과 랙 피니언 모드 전환
- 모듈 또는 목표 외경 기준의 파라메트릭 크기 제어
- 압력각, 잇수, 구멍 지름, 백래시 입력
- 사용자 지정 원형 구멍, TT 모터 Double-D 축, N20 모터 3mm D축 구멍 프리셋
- 5mm 그리드, 하단 그리드 범례, 회전 시뮬레이션
- 접이식 맞물림 제어판에서 RPM, 백래시, 리셋 제어
- 별도 내보내기 메뉴에서 SVG/DXF/STL 저장
- 내보내기 부품 선택, 부품 간격, STL 두께 설정

## 설계 기준과 제한

- 모든 치수 단위는 mm입니다.
- 평기어 치형은 표준 인벌류트 근사 형상입니다.
- 표준 평기어 외경은 `모듈 × (잇수 + 2)` 관계를 따릅니다. 외경 기준 설계 시 이 식으로 공통 모듈을 역산합니다.
- 맞물리는 평기어 쌍은 같은 모듈을 공유해야 하므로, 기어 1 외경 또는 기어 2 외경 중 하나를 기준으로 전체 크기를 정합니다.
- 랙 길이는 잇수 단위의 정수로 처리합니다. 소수 길이는 생성 치형 수와 실제 길이가 달라질 수 있어 입력 단계에서 정수로 보정합니다.
- 백래시는 치형 자체를 깎는 값이 아니라 두 기어 또는 피니언-랙의 중심 거리를 넓히는 맞물림 간극입니다.
- N20 모터 프리셋은 일반적인 3mm D축 기준입니다. 제조사별 평면부 치수는 조금씩 다를 수 있으므로 출력 전 실제 모터 축을 캘리퍼스로 확인하는 것이 좋습니다.
- DXF는 가벼운 `LWPOLYLINE`/`CIRCLE` 중심의 2D 형식입니다. CAD/CAM 도구별 열림 상태는 실제 장비 흐름에서 한 번 더 확인해야 합니다.
- STL은 2D 치형을 설정한 두께만큼 직선 돌출한 ASCII STL입니다. 축 구멍은 관통 구멍으로 반영됩니다.
- 여러 부품을 함께 내보낼 때는 맞물림 위치가 아니라 출력용 분리 배치로 저장합니다. 부품 간격은 내보내기 설정에서 조정합니다.
